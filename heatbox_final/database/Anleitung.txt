Bitte führen Sie das Script auf Datenbankebene aus.
Dazu können Sie den GESAMTEN Inhalt in eine Befehlszeile kopieren und ausführen.

Der Anlegevorgang kann mehrere Minuten dauern.
Danach können Sie die HeatBox Website ohne probleme auf Ihrem localhost verwenden.

Bitte beachten Sie, dass Sie die Befehlszeile mit einem User ausführen, der sowohl Datenbanken als auch User erstellen darf.
